# Plug-and-Play ASO Decision Demo (Pain/DRG)

Streamlit app for gene prioritization and fast ASO design.
- Home, SpinalCord Specificity, Pain Genetics, DRG markers, Shortlist, ASO Design
- GFF-only fast isoform schematic; arc/circle structure and pairing-depth

## Local run
pip install -r requirements.txt
streamlit run app.py
